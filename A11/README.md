## 🔑 重點摘要

- **核心主張**：提出「人工知識生成（AKM）」概念，分析 GenAI 如何以 SECI 模型框架重塑知識管理流程
- **研究方法**：文獻計量分析 1,411 篇論文（描述性分析 + 科學知識圖譜），是迄今最大規模的 GenAI × KM 系統性回顧之一
- **新概念維度**：提出「機器維度（Machine Dimension）」延伸傳統知識生成模型（原本只有個人/群組/組織三層）
- **SECI 框架延伸**：明確以 SECI 模型分析 GenAI 如何介入知識創造的四個轉化環節
- **實務工具**：提出「GenAI 就緒性分類法（taxonomy）」供管理者與實務工作者使用
